# Commit Workflow

You will follow the below steps during this commit stage:

1. review unit tests to make sure adequate coverage and success of all tests throughout the project
2. make sure that the PROGRESS.md and PROJECT_PLAN.md have been updated to reflect the current state
3. review and update any relevant documentation
4. commit and push to github
5. update the indexes using your code-intel tool
6. confirm the stats of the code-intel indexes
